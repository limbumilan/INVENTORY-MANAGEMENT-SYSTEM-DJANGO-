# 📦 Inventory Management System (Django)

A web-based Inventory Management System built with Django that helps businesses manage products, stock levels, suppliers, and inventory operations efficiently.

---

## 🚀 Project Overview

This system is designed to simplify inventory tracking and management for small to medium businesses. It provides a centralized dashboard to manage products, suppliers, and stock movement with user authentication and role-based access.

---

## ✨ Features

- 🔐 User authentication (Login / Logout)
- 👤 Role-based access (Admin / Staff)
- 📦 Product management (Add, Update, Delete)
- 📊 Stock management and tracking
- 🏢 Supplier management system
- 📁 Category management
- 📈 Dashboard with summary statistics
- 🔍 Search and filter products
- 📄 Inventory reporting system

---

## 🛠️ Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Database:** SQLite (default) / PostgreSQL (optional)
- **Authentication:** Django built-in auth system
